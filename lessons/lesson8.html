Lesson 8: Memory and Learning Systems

üß† Core Question

How does the brain store and retrieve information? What makes some memories last a lifetime while others fade in seconds?

Types of Memory Systems

Working Memory

Function: Temporary storage and manipulation of information

  • Duration: Seconds to minutes
  • Capacity: 7¬±2 items (Miller's Law)
  • Mechanism: Persistent neural activity in prefrontal cortex
  • Example: Remembering a phone number while dialing

Long-term Memory

Function: Permanent storage of information

  • Duration: Minutes to lifetime
  • Capacity: Virtually unlimited
  • Mechanism: Synaptic weight changes and new connections
  • Types: Declarative (explicit) and procedural (implicit)

Episodic Memory

Function: Memory for specific events and experiences

  • Features: Time, place, context, emotions
  • Brain region: Hippocampus and associated structures
  • Formation: Requires conscious attention and encoding
  • Example: Remembering your first day of school

Neural Mechanisms of Memory

Synaptic Plasticity

The biological basis of learning and memory:

  • Long-term Potentiation (LTP): Strengthening of synaptic connections
  • Long-term Depression (LTD): Weakening of synaptic connections
  • Spike-timing Dependent Plasticity (STDP): Timing-based learning rules
  • Homeostatic plasticity: Maintaining overall network stability

üí° Hebb's Rule

"Neurons that fire together, wire together." This principle, first proposed by Donald Hebb, explains how associations are formed in neural networks.

Memory Consolidation

The process of converting temporary memories into permanent ones:

  • Initial encoding: Hippocampus rapidly binds information
  • Systems consolidation: Gradual transfer to cortical areas
  • Reconsolidation: Memories become labile when recalled
  • Sleep role: Memory replay during slow-wave sleep

The Hippocampal Memory System

Structure and Function

  • Dentate gyrus: Pattern separation for distinct memories
  • CA3 region: Associative network for pattern completion
  • CA1 region: Comparator and output to cortex
  • Entorhinal cortex: Interface between hippocampus and cortex

Place Cells and Grid Cells

Specialized neurons that encode spatial information:

  • Place cells: Fire when animal is in specific location
  • Grid cells: Create hexagonal firing patterns for navigation
  • Border cells: Respond to environmental boundaries
  • Time cells: Encode temporal sequences and intervals

üèÜ Nobel Prize 2014

John O'Keefe, May-Britt Moser, and Edvard Moser won the Nobel Prize for discovering place cells and grid cells, revealing how the brain creates spatial memories and navigation systems.

Memory Disorders and Insights

Alzheimer's Disease

  • Early symptoms: Hippocampal damage affects new memory formation
  • Protein aggregation: Amyloid plaques and tau tangles disrupt synapses
  • Network breakdown: Loss of connectivity between brain regions
  • Research target: Understanding synaptic dysfunction

Amnesia Studies

  • H.M. (Henry Molaison): Lost ability to form new episodic memories
  • Anterograde amnesia: Cannot form new memories after brain damage
  • Retrograde amnesia: Loss of memories before brain damage
  • Preserved skills: Procedural memory often intact

Computational Models of Memory

Hopfield Networks

Mathematical model of associative memory:

  • Attractor dynamics: Memories as stable states
  • Pattern completion: Partial cues retrieve full memories
  • Capacity limits: Number of memories that can be stored
  • Spurious states: False memories from pattern interference

Temporal Difference Learning

Algorithm for learning predictions and associations:

  • Prediction error: Difference between expected and actual outcomes
  • Dopamine system: Implements prediction error signaling
  • Reinforcement learning: Learning from rewards and punishments
  • Value functions: Estimating future rewards

Memory in Brain Emulation

Technical Challenges

  • Multi-scale plasticity: From molecules to circuits
  • Timing precision: Millisecond-level dynamics
  • State dependence: Memory affected by brain state
  • Reconstruction fidelity: Preserving synaptic weights

Philosophical Questions

  • Memory identity: Are you the sum of your memories?
  • False memories: What if emulation includes false memories?
  • Selective preservation: Which memories are essential?
  • Memory modification: Could traumatic memories be edited?

üî¨ Research Implications

Understanding memory systems is crucial for brain emulation. An emulated brain without proper memory formation and retrieval would lack the continuity and learning that define personal identity.

Learning Algorithms

Supervised Learning

  • Error-driven: Learn from correct examples
  • Backpropagation: Artificial version in deep learning
  • Teacher signal: External feedback guides learning
  • Generalization: Applying learning to new situations

Unsupervised Learning

  • Statistical learning: Discovering patterns in data
  • Hebbian plasticity: Correlation-based learning
  • Sparse coding: Efficient representation learning
  • Self-organization: Spontaneous pattern formation

Key Takeaways

  1. Multiple memory systems serve different functions
  2. Synaptic plasticity is the biological basis of learning
  3. The hippocampus is crucial for episodic memory formation
  4. Memory consolidation transfers information from temporary to permanent storage
  5. Understanding memory is essential for brain emulation
